Welcome to the **Mastering Git & GitHub with Laravel** course! This repository provides a **complete 2-week, instructor-ready curriculum** designed to help learners become proficient with Git, GitHub, and professional collaboration workflows in Laravel projects. The course includes **modular lessons, practical exercises, rubrics, and team projects** to reinforce real-world version control practices.

---

## **Getting Started**

1. Review the high-level course map in [`syllabus.md`](syllabus.md).  
2. Progress sequentially through the lessons: `course/week1` → `course/week2`. Each day includes theory, exercises, and small projects.  
3. For questions or assignment submissions, use the **course Discord server** (see Submission section).

> **Pro Tip:** Clone the repo using HTTPS for compatibility, or SSH if you prefer secure key-based authentication.

## **Working Through Lessons**

- Read the lesson file (`course/week*/day*.md`) carefully.  
- Perform exercises locally using the terminal and Git (HTTPS recommended).  
- Use **feature branches** for exercises and submit **Pull Requests** to practice collaborative workflows.  

**Exercise Example:**  
- `students.md` — used in Week 2 Day 8 for PR exercises.

## **Submission Guidelines**

- All assignments must be submitted to the course **Discord server**, in the `#git-and-github` channel.  
- **Invite Link:** [https://discord.gg/wcFzwswv](https://discord.gg/wcFzwswv)

**Why Discord?**  
- Centralized, real-time feedback.  
- Easy tracking for instructors and peer review.  

---

## **Grading & Rubrics**

- Daily assignments include rubrics (typically 100 pts) specifying required deliverables: **repo links, PR links, branch names**.  
- Weekly projects have detailed rubrics in the project files.  
- For team projects, **one member submits** the final project and tags teammates.
